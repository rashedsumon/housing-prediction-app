# Housing Price Prediction Streamlit Web App

An AI-driven real estate valuation tool built using Python, Scikit-Learn, and Streamlit. The application downloads data via `kagglehub`, trains a regression model on the fly if needed, and deploys natively to Streamlit Cloud.
